# Support

Welcome to the Syl ecosystem! Here's how to get help.

## Documentation

Start with the [Syl Docs](https://syllang.com) — the official language
documentation, reference, and guides.

## GitHub Issues

- **Bug reports** — use the [Bug Report template](https://github.com/syllang/docs/issues/new?template=bug_report.md)
- **Feature requests** — use the [Feature Request template](https://github.com/syllang/docs/issues/new?template=feature_request.md)

Before opening a new issue, please search [existing issues](https://github.com/syllang/docs/issues)
to avoid duplicates.

## GitHub Discussions

For questions, ideas, and general conversation, join our
[GitHub Discussions](https://github.com/syllang/docs/discussions).
