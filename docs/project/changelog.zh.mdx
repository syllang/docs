# 更新日志

本文记录 Syl 编译器的版本变更。每个版本列出新增功能、变更和 bug 修复。

---

## 0.1.0（最新）

### 新增

- 编译器核心流水线：前端 → 语义分析 → 展开 → 后端
- `module` 和 `cell` 声明
- `interface` 和 `view` 接口系统
- 泛型参数 `Nat`、`Domain` 和类型参数
- `reg` 寄存器和 `next` 时序赋值
- `map` 组合映射
- `fn` 编译期函数
- 编译期 `if` 和 `for`
- `select` 和 `match` 表达式
- `bundle` 和 `enum` 复合类型
- `place` 模块实例化
- `use` 导入系统
- 标准库：Stream、Stage、Pipeline
- 架构合约测试套件
- 一致性测试套件
- Verilator 烟雾测试

### 已知限制

- 不支持有符号整数
- 不支持浮点数
- 不支持多时钟域的综合检查
- LSP 支持为基本级别
