# 词法语法

本文定义 Syl 的词法规则。它服务于编译器实现者和需要精确理解语言的用户。

---

## 字符集

Syl 源代码使用 UTF-8 编码。有效字符包括：

- ASCII 字母：`a-z`、`A-Z`
- 数字：`0-9`
- 下划线：`_`
- 运算符和分隔符：`+ - * / % = ! < > & | ^ ~ . , : ; ( ) [ ] { } @ #`
- 空白：空格、制表符、换行符

注释使用 `//` 开头，直到行尾。

## 关键字

Syl 的关键字列表。关键字不能用作标识符。

```
cell  extern  fn  map  const  let  var  signal  reg
in  out  inout  next  place  return
if  else  for  while
select  priority  unique  match
enum  bundle  interface  view
use  package
true  false
and  or  not  xor  eq
Domain  Nat  Bool  Bit  Clock  Reset
compile_error
```

## 标识符

标识符以字母或下划线开头，后跟字母、数字或下划线。

```
counter  data_in  FIFO  _internal  Stream_8
```

标识符区分大小写。`counter` 和 `Counter` 是不同的标识符。

## 字面量

### 整数字面量

```
42       // 十进制
0xFF     // 十六进制
0b1010   // 二进制
```

整数字面量可以是十进制、十六进制或二进制。数字中可以使用下划线分隔：`1_000_000`。

### 布尔字面量

```
true    false
```

### 字符串字面量

```
"hello"   "data"
```

字符串用于编译期诊断信息。

## 运算符

### 硬件运算符

```
and   or   xor   not   eq       // 逻辑/按位运算
+     -    *     /     %        // 算术
<<    >>                        // 移位
==    !=    <    >    <=   >=   // 比较
:=                               // 连续赋值
```

### 编译期运算符

```
and   or   not                  // 逻辑运算
+     -    *     /     %        // 算术
<<                              // 移位
==    !=    <    >    <=   >=   // 比较
```

## 分隔符

```
( )    // 分组和参数列表
{ }    // 代码块
[ ]    // 数组访问和泛型参数
:      // 类型声明
,      // 分隔
.      // 字段访问和枚举变体
;      // 语句结束（可选）
->     // 返回值声明
=>     // 选择/匹配分支
:=     // 连续赋值
```

## 注释

```
// 这是单行注释
```

Syl 只支持单行注释。

## 空白

空白（空格、制表符、换行符）用于分隔 Token。Syl 不依赖缩进，但建议使用一致的缩进风格。
