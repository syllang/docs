# sylc 编译器

`sylc` 是 Syl 编译器的命令行入口。它将 Syl 源代码编译为 SystemVerilog 代码。本文说明编译器的使用方法、工作流程和输出格式。

---

## 基本使用

编译一个 Syl 文件：

```
sylc input.syl
```

编译多个文件：

```
sylc top.syl counter.syl types.syl
```

输出到文件：

```
sylc input.syl -o output.sv
```

## 编译流程

`sylc` 执行以下步骤：

1. **解析。** 读取源文件，解析为 AST。
2. **语义分析。** 名字解析、类型检查、能力检查。
3. **展开。** 将类型化描述展开为硬件结构。
4. **后端。** 将硬件结构输出为 SystemVerilog。

如果任何步骤检测到错误，编译器输出诊断信息并退出。

## 输出

### SystemVerilog 代码

编译成功后，SystemVerilog 代码输出到标准输出或指定文件。每个 Syl `module` 对应一个 SystemVerilog `module`。

```verilog
// Generated by sylc.

module AddOne(
    input  [7:0] x,
    output [7:0] y
);
    assign y = x + 8'd1;
endmodule
```

### 诊断信息

诊断信息输出到标准错误。每条诊断包含：

- 严重级别：error 或 warning
- 错误代码：如 `E_MIDDLE_TYPE_MISMATCH`
- 描述信息：说明问题
- 源码位置：文件名、行号、列号

## 选项参考

| 选项 | 说明 |
|------|------|
| `-o <file>` | 输出到文件 |
| `--std <dir>` | 标准库路径 |
| `--help` | 显示帮助 |
| `--version` | 显示版本 |
